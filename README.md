## Nocturne Music Player

![alt text](./screen-shoots/mockup-nocturne.png)
Nocturne adalah aplikasi pemutar musik desktop berdesain premium dengan pendekatan offline-first. Semua fungsi inti (memutar file lokal, playlist, equalizer, lirik tersinkron, visualizer) berjalan penuh tanpa koneksi internet. Aplikasi ini dirancang untuk audiophile dan kolektor musik lokal, dengan tambahan fitur pemutaran sumber online (SoundCloud) sebagai pelengkap opsional.
